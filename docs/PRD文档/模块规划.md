#### 3.6.3 规则类型设计

```java
public enum RuleType {
    /**
     * 完整性规则 - 检测字段是否为空
     */
    COMPLETENESS("完整性", "检测字段是否为空"),

    /**
     * 唯一性规则 - 检测字段值是否唯一
     */
    UNIQUENESS("唯一性", "检测字段值是否唯一"),

    /**
     * 格式规则 - 检测字段值是否符合指定格式
     */
    FORMAT("格式", "检测字段值是否符合指定格式"),

    /**
     * 值域规则 - 检测字段值是否在指定范围内
     */
    VALUE_RANGE("值域", "检测字段值是否在指定范围内"),

    /**
     * 一致性规则 - 检测跨表关联一致性
     */
    CONSISTENCY("一致性", "检测跨表关联一致性"),

    /**
```
