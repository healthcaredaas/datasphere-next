### 1.2 当前技术栈版本

| 组件 | 当前版本 | 目标版本 |
|------|----------|----------|
| Spring Boot | 3.4.6 | **3.5.11** |
| Spring Cloud | 2024.0.1 | **2025.0.0** |
| Spring Cloud Alibaba | 2023.0.3.3 | **2025.0.0** |
| Spring Security OAuth2 | 1.4.3 | 1.5.x |
| MyBatis Plus | 3.5.12 | 3.5.12 (保持) |
| JDK | 17 | 17 (保持) |

### 1.3 现存问题分析

1. **模块划分冗余**: daas-tenant 多租户模块包含4个子模块，与 daas-web-rest-tenant 形成交叉依赖
2. **功能耦合**: 多租户功能渗透到数据层、Web层、安全层，增加系统复杂度
3. **版本滞后**: Spring Cloud Alibaba 2023.0.3.3 较旧，需升级至 2025.0.0
4. **依赖复杂**: daas-cloud 过度依赖 daas-boot 的租户相关模块

## 二、重构目标

### 2.1 核心目标

| 目标 | 说明 | 验收标准 |
|------|------|----------|
| 版本升级 | 升级至指定版本 | 所有组件版本符合要求，编译通过 |
| 去多租户 | 完全移除多租户功能 | 无租户相关代码、表、依赖 |
| 模块优化 | 精简模块结构 | 模块数量减少，职责清晰 |
| 功能保持 | 保留原有业务功能 | 非租户相关功能正常运行 |

### 2.2 重构原则

1. **最小侵入原则**: 除多租户相关代码外，不修改原有业务逻辑
2. **渐进式重构**: 先移除租户模块，再优化模块结构
3. **版本兼容性**: 确保升级后的版本相互兼容
4. **可验证性**: 每个步骤都可编译、可测试

### 5.2 实体类继承关系调整

#### 调整前:
```java
// 继承多租户实体
public class RbacUser extends TenantEntity {
    // 自动包含 tenant_id 字段
}

public class RbacResource extends TenantTree {
    // 树形结构 + 租户
}
```

#### 调整后:
```java
// 继承普通基础实体
public class RbacUser extends BaseEntity {
    // 无 tenant_id 字段
}

public class RbacResource extends BaseTreeEntity {
    // 纯树形结构
}
```

### 5.3 数据源配置调整

#### 调整前:
```yaml
spring:
  datasource:
    dynamic:
      primary: default
      datasource:
        default:
          driver-class-name: com.mysql.cj.jdbc.Driver
          url: jdbc:mysql://localhost:3306/cloud_daas
```

#### 调整后:
```yaml
spring:
  datasource:
    driver-class-name: com.mysql.cj.jdbc.Driver
    url: jdbc:mysql://localhost:3306/cloud_daas
    username: ${DB_USERNAME}
    password: ${DB_PASSWORD}
```

### 5.4 Redis配置调整

#### 调整前:
- 使用 `DynamicRedissonConnectionFactory` 动态创建Redis连接
- 租户隔离通过不同Redis DB或实例实现

#### 调整后:
- 使用标准 `RedissonConnectionFactory`
- 单Redis实例，无租户隔离

## 七、风险与注意事项

### 7.1 版本兼容性风险

| 风险点 | 影响 | 缓解措施 |
|--------|------|----------|
| Spring Boot 3.5.x 兼容性 | 部分组件可能不兼容 | 提前验证各组件版本兼容性 |
| Spring Cloud Alibaba 2025 | 配置属性可能有变化 | 参考官方迁移文档 |
| MyBatis Plus 兼容性 | 与Spring Boot 3.5兼容性 | 使用最新稳定版本 |

### 7.2 功能影响分析

| 功能模块 | 影响程度 | 说明 |
|----------|----------|------|
| 用户管理 | 低 | 仅移除tenant_id字段 |
| 角色权限 | 低 | 仅移除tenant_id字段 |
| 认证授权 | 中 | 需移除租户过滤器配置 |
| 数据字典 | 低 | 仅移除tenant_id字段 |
| 审计日志 | 低 | 仅移除tenant_id字段 |
| 多租户配置 | 高 | 完全移除 |
